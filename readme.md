# Lab work 2: Own cat program
Authors (team): Viktor Povazhuk

## Prerequisites

Used libraries:
* boost

### Compilation

Run:
```bash
$ ./compile.sh -O
```

### Usage

Run:
```bash
$ ./mycat [-A] [-h] file1 file2 ... 
```

To get detailed description of options, run:
```bash
$ ./mycat -h
```

### Results

Learned:
* system calls to read from file and write into console
* repeated options parser usage
